ilcli’s documentation¶

ilcli: I like command-line interfaces.

A helper library for creating CLIs quickly with a few cool features. Take a look to the Quick Start section.

Installation¶

You can use the following pip command for installing ilcli in your environment:

$ pip install ilcli
